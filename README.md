# Aerospace Day Activity — SGT

A browser-based simulations for a 35–40 minute aerospace session. No install, no login, no coding required. Works in any modern browser (Chrome, Firefox, Edge).


## Mission Design Challenge (Grades 6-9)
**File:** `mission-design.html`

A live-calculation tool (like a spreadsheet) where students design a spacecraft mission to the Moon, Mars, Europa, or Titan. All results update instantly as sliders change.

**Concepts covered:** Tsiolkovsky rocket equation · Specific impulse (Isp) · Mass ratio · Delta-v budget · Mission trade-offs

Includes three built-in challenge tasks: Essential / Advanced / Expert.

## How to use

### Option A — GitHub Pages (recommended, gives you a shareable URL)
1. Upload both `.html` files to this repository
2. Go to **Settings → Pages**
3. Under "Source", select **Deploy from a branch → main → / (root)**
4. Click Save — GitHub will give you a URL like `https://yourusername.github.io/repo-name/`
5. Share these links with students:
   - Activity: `https://yourusername.github.io/repo-name/activity2-mission-design.html`

### Option B — Download and open locally
1. Download the `.html` files
2. Double-click to open in any browser
3. Share via USB drive, email, or network folder
